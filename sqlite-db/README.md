# @csgaglobal/sqlite-db

SQLite database operations with query execution, schema management, data import/export, and database analysis.

**Category:** Database | **Version:** 1.0.0 | **License:** MIT

## Overview

Part of the [CSGA Global MCP Server Platform](https://csga-global.org/) — a comprehensive suite of 70+ Model Context Protocol servers for AI governance, cybersecurity, and enterprise tooling.

## Features

- In-memory & file-based databases
- Query execution & analysis
- Data import/export (CSV, JSON)
- Schema introspection

## Installation

```bash

```
