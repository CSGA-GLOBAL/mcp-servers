CSV data analysis with statistical computing, pattern detection, data transformation, and visualization generation.

**Category:** Data & Analytics | **Version:** 1.0.0 | **License:** MIT

## Overview

Part of the [CSGA Global MCP Server Platform](https://csga-global.org/) — a comprehensive suite of 70+ Model Context Protocol servers for AI governance, cybersecurity, and enterprise tooling.

## Features

- Statistical analysis & aggregation
- Pattern detection algorithms
- Data transformation pipelines
- Visualization generation
